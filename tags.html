HTML Tags

HTML tags are the building blocks of web pages. Below are some commonly used tags:

  • <h1> to <h6> - Headings
  • <p> - Paragraph
  • <a> - Anchor (links)
  • <img> - Image
  • <ul>, <ol>, <li> - Lists
  • <div>, <span> - Containers
<h1>This is a heading</h1>
<p>This is a paragraph.</p>

HTML Attributes

Attributes provide additional information about an element.

  • href - Specifies the URL for an anchor tag.
  • src - Specifies the source for an image.
  • alt - Provides alternative text for an image.
  • class - Specifies a class name for styling.
  • id - Specifies a unique ID for an element.
<a href="https://example.com">Visit Example</a>
<img src="image.jpg" alt="Sample Image">

CSS Properties

CSS is used to style HTML elements. Here are some commonly used properties:

  • color - Sets the text color.
  • background-color - Sets the background color.
  • font-size - Sets the font size.
  • margin - Sets the space outside an element.
  • padding - Sets the space inside an element.
p {
    color: blue;
    background-color: lightgray;
    font-size: 16px;
    margin: 10px;
    padding: 5px;
}
